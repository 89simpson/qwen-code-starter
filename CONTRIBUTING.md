# Contributing to Qwen Code Starter

Thank you for your interest in contributing to Qwen Code Starter! This document provides guidelines and instructions for contributing.

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/qwen-code-starter.git`
3. Create a branch: `git checkout -b feature/your-feature-name`
4. Make your changes
5. Run tests: `python tests/test-merge-qwen-md.py`
6. Commit and push: `git push origin feature/your-feature-name`
7. Open a Pull Request

## Development Guidelines

### Code Style

- Use clear, descriptive variable and function names
- Keep functions small and focused
- Add docstrings for public functions
- Include inline comments for complex logic

### Testing

- All changes must include tests
- Run existing tests before submitting: `python tests/test-merge-qwen-md.py`
- Ensure test coverage is maintained or improved

### Documentation

- Update README.md for user-facing changes
- Update CHANGELOG.md with your changes
- Document new templates, rules, skills, or agents

## Pull Request Process

1. **Create PR**: Open a Pull Request with a clear title and description
2. **Testing**: Ensure all tests pass
3. **Review**: Wait for code review from maintainers
4. **Address Feedback**: Make requested changes
5. **Merge**: Once approved, a maintainer will merge your PR

## What to Contribute

### Templates

New project templates for different types of projects (code, content, hybrid).

### Rules

New rules for specific use cases or industries.

### Skills

New reusable workflows and capabilities.

### Agents

New specialized AI personas for specific domains.

### Hooks

New automation scripts for additional events.

### Documentation

Improvements to existing documentation, examples, or guides.

### Bug Fixes

Fix any issues found in the framework.

## Reporting Issues

- Use GitHub Issues to report bugs or suggest features
- Include steps to reproduce the issue
- Provide expected vs actual behavior
- Include relevant logs or screenshots

## Questions?

If you have questions, please open an issue for discussion.

## Code of Conduct

- Be respectful and inclusive
- Provide constructive feedback
- Accept constructive criticism
- Focus on what's best for the community

Thank you for contributing to Qwen Code Starter!
