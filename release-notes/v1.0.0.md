# Release Notes for v1.0.0

## Summary

Initial release of Qwen Code Starter — a complete adaptation of Claude Code Starter specifically designed for Qwen Code.

## What's New

### Core Framework

- **Project Type Detection**: Automatic detection of code, content, and hybrid projects
- **QWEN.md System**: Project passport with comprehensive templates
- **Modular Architecture**: Rules, skills, agents, and hooks as separate components
- **Global Installation**: Support for `~/.qwen/` global layer
- **Migration Tool**: Convert Claude Code Starter projects to Qwen Code

### Templates

#### Code Projects
- **QWEN.md**: Complete project passport template
- **7 Rules**: autonomy, delegation, context-management, production-safety, local-first, logging, commit-policy
- **6 Skills**: start, finish, testing, db-migrate, housekeeping, research
- **3 Agents**: researcher, implementer, reviewer

#### Content Projects
- **QWEN.md**: Content-focused project passport
- **5 Rules**: content-pipeline, content-quality, source-management, content-formats, content-commit-policy
- **7 Skills**: research, outline, write-content, review-content, enrich, content-index, housekeeping
- **4 Agents**: researcher, writer, editor, reviewer
- **5 Content Templates**: chapter, lesson, article, document, transcript

### Installation

- **Single-file installer**: `init-project.sh` with auto-detection
- **Global installer**: `install-global.sh` for `~/.qwen/` setup
- **Migration script**: `migrate.sh` for existing projects
- **Mode switcher**: `switch-repo-access.sh` for permissions

### Hooks System

- **PreCompact**: Before context compaction
- **PostToolUse**: After successful tool execution
- **PostToolUseFailure**: After tool failure
- **SubagentStop**: When subagent completes

### Utilities

- **merge_qwen_md.py**: Safe QWEN.md merging with conflict detection
- **20 unit tests**: Full test coverage for merge utility
- **install_common.sh**: Shared installation library

## Qwen Code Adaptations

| Component | Claude Code | Qwen Code |
|-----------|-------------|-----------|
| Config dir | `.claude/` | `.qwen/` |
| Passport | `CLAUDE.md` | `QWEN.md` |
| Permissions | `manifest.md` | `settings.json` + `.qwenignore` |
| Agents | Nested dirs | Flat (`agents/name.md`) |
| Skills | With `allowed-tools` | Without `allowed-tools` |
| Hooks | 4 types | 4 types (no PostCompact) |

## Installation

### Quick Start

```bash
# New project
curl -fsSL https://raw.githubusercontent.com/qwen-code-starter/main/scripts/init-project.sh | bash

# Or clone and run
git clone https://github.com/qwen-code-starter/qwen-code-starter.git
cd qwen-code-starter
./init-project.sh
```

### Migration

```bash
# Migrate existing Claude Code Starter project
./scripts/migrate.sh /path/to/existing/project
```

## Testing

```bash
python tests/test-merge-qwen-md.py
```

## Known Limitations

1. **No PostCompact hook**: Qwen Code doesn't support this event
2. **No allowed-tools**: Skills don't support tool restrictions in frontmatter
3. **Project type detection**: Uses heuristics, may need manual override
